springboot+mybabits+mysql
1，添加Mybatis的起步依赖,
<!--mybatis起步依赖-->
<dependency>
<groupId>org.mybatis.spring.boot</groupId>
<artifactId>mybatis-spring-boot-starter</artifactId>
<version>1.1.1</version>
</dependency>

2， 添加数据库驱动坐标
<!-- MySQL连接驱动 -->
<dependency>
<groupId>mysql</groupId>
<artifactId>mysql-connector-java</artifactId>
</dependency>
3，在application.properties中添加数据量的连接信息
spring.datasource.driverClassName=com.mysql.jdbc.Driver
spring.datasource.password=root
mybatis.type-aliases-package=com.demo
mybatis.mapper-locations=classpath:mapper/*Mapper.xml
4，
     创建user表
CREATE TABLE `user` (  
`id` INT(11) NOT NULL AUTO_INCREMENT,  
`username` VARCHAR(50) DEFAULT NULL,  
`password` VARCHAR(50) DEFAULT NULL,  
`name` VARCHAR(50) DEFAULT NULL,  
PRIMARY KEY (`id`) )
ENGINE=INNODB AUTO_INCREMENT=10 DEFAULT CHARSET=utf8

public class User{...};

public interface Usermapper { List<User> queryUserList();}
@Mapper注解的的作用
1:为了把mapper这个DAO交給Spring管理    @@Autowired就可以调用了
2:为了不再写mapper映射文件（？？？）
3:为了给mapper接口 自动根据一个添加@Mapper注解的接口生成一个实现类（意思是我可以把这个Usermapper方法直接当作是实现类调用）

我这里建了一个usermapper接口，里面加了@mapper。
又在resouses里面建了一个，usermapper.xml。<mapper></mapper>。
usermapper接口靠@mapper定位到xml里的<mapper></mapper>过程叫做映射。
